  • Rank of a matrix: the number of non-zero pivots after completing elimination
  • To find a vector in the nullspace: complete elimination and identify the pivot columns and free columns. Give each free column the value of 1, 0, .. or 0, 1, .. and then substitute these values in to solve for the pivot column values. The solution is any linear combination of these special solution vector(s).
  • The rank is the number of pivot columns, so the total number of columns minus the rank gives the number of free columns, which is the number of columns which when multiplied by a scalar and added together makes up the nullspace
  • Reduced row echelon form (R): after completing elimination, (having an upper-triangular matrix), do elimination upwards so that there as many 0s as possible both above and below the pivots, and make all of the pivots 1 by dividing

    # Rank of a Matrix
     
    A = np.array([[1, 2, 2, 2], [0, 0, 2, 4], [0, 0, 0, 0]])
    print(f'A = \n{A}\nRank of A = {np.linalg.matrix_rank(A)}')
     
    #OUTPUT:
    '''
    A =
    [[1 2 2 2]
     [0 0 2 4]
     [0 0 0 0]]
    Rank of A = 2
    '''
