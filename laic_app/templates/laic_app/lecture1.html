  • The fundamental problem of Linear Algebra: solving systems of linear equations
  • Matrix: A rectangular array of numbers
  • Any system of n linear equations with n unknowns can be solved using a Row Picture, Column Picture, or Matrix form
  • Row picture: Draw each equation on the coordinate plane and find where they intersect. This is only useful in the case of 2 equations with 2 unknowns, as finding the intersection in 3 dimensions or higher is not trivial.
  • Column picture: Stack the equations on top of each other and align the variables. Each variable is associated with a 1D column matrix (vector). Draw each of these vectors on the coordinate plane - a specific linear combination (multiplying and adding) of these column vectors generates the solution vector.
  • Matrix form: Stack the equations on top of each other and align the variables. The matrix form is Ax = b, where A is a matrix of all the coefficients, x is a 1D column matrix (vector) of the variables, and b is a 1D column matrix (vector) of the solution.
  • Ax = b can be solved for every b as long as the matrix A is non-singular / invertible (as long as each column vector is linearly independent, which means none of the column vectors are a linear combination of the others) AND the number of equations equals the number of unknowns

    import numpy as np
    from sympy import linear_eq_to_matrix, symbols
    import ast
     
    # System of 2 equations of the form A*x + B*y - C = 0
     
    x, y = symbols('x y')
    eqns = [2*x - y,
            -x + 2*y - 3] # = 0 is implied for these equations
     
    tempA1, tempb1 = linear_eq_to_matrix(eqns, [x, y])
     
    #converts MutableDenseMatrix returned by linear_eq_to_matrix to list that can be used in numpy array
    def getAandb():
        tempA2 = str(tempA1); tempb2 = str(tempb1)
        tempA2 = tempA2[7:len(tempA2)-1]; tempb2 = tempb2[7:len(tempb2)-1]
        tempA2 = ast.literal_eval(tempA2); tempb2 = ast.literal_eval(tempb2)
        return tempA2, tempb2
     
    # Now we use NumPy to solve for x in Ax = b, plugging in A and b from above
     
    def getSolution():
        A = np.array(getAandb()[0])
        b = np.array(getAandb()[1])
        x = np.linalg.solve(A, b)
        for i in range(0, len(x)):
            if x[i] == -0:
                x[i] = 0
        return x
     
    print(f'\nA solution to the system of 2 equations {eqns} is: \n{getSolution()} \n\n')
     
    #OUTPUT:
    '''
    A solution to the system of 2 equations [2*x - y, -x + 2*y - 3] is:
    [[1.]
     [2.]]
    '''
