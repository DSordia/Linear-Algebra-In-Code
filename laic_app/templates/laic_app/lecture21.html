  • Eigenvector: take a matrix A and a vector x. If the vector Ax points in the same direction as x, Ax is an eigenvector, which implies Ax is some multiple of x. Ax = λx. In this case, λ is the eigenvalue
  • The eigenvectors with eigenvalue λ = 0 are the vectors in the nullspace (only when A is non-invertible/singular
  • The eigenvalues for a projection matrix: λ = 1 for any x in the plane, λ = 0 for any x perpendicular to the plane (in the nullspace)
  • n x n matrices have n eigenvalues
  • The sum of all the eigenvalues = the sum across the diagonal of the matrix (which is the trace of the matrix)
  • The product of all the eigenvalues = the determinant of the matrix
  • How to solve Ax = λx: First, rewrite as (A-λI)x = 0. This implies that A - λI is singular and det(A - λI) = 0. A - λI is the matrix A with λ subtracted from each value of the diagonal. Once you solve for λ, plug each λ into A - λI and the corresponding eigenvector is the vector for which A - λI = 0
  • Take a matrix A and scalar c. If Ax = λx, then (A + cI)x = (λ+c)x, so the eigenvalues change by c but the eigenvectors remain the same
  • The eigenvalues of a matrix A plus or times the eigenvalues of a matrix B are not necessarily the eigenvalues of A+B or A*B (since A and B have different eigenvectors)
  • The eigenvalues of a triangular matrix are the values of the diagonal
  • The possibilities for eignvalues: real, complex, or repeated roots (repeated roots only generates 1 eigenvector)

    import numpy as np
     
    # Eigenvalues/eigenvectors of a projection matrix
     
    A = np.array([[0, 1], [0, 1]])
    evalues, evectors = np.linalg.eig(A)
     
    print(f'A = \n{A}\nEigenvalues of A = {evalues}\nEigenvectors of A = \n{evectors}')

    # Showing that the eigenvalues of a matrix A plus or times the eigenvalues of a matrix B are not necessarily the eigenvalues of A+B or A*B
     
    A = np.array([[3, 1], [1, 3]])
    B = np.array([[1, 2], [-1, 4]])
    AplusB = A + B
    AtimesB = A * B
    evalues, evectors = np.linalg.eig(A)
    evalues2, evectors2 = np.linalg.eig(B)
    evalues3, evectors3 = np.linalg.eig(AplusB)
    evalues4, evectors4 = np.linalg.eig(AtimesB)
     
    print(f'Eigenvalues of A = {evalues}\nEigenvalues of B = {evalues2}\nEigenvalues of A+B = {evalues3}\nEigenvalues of A + Eigenvalues of B = {evalues + evalues2}\n'
    f'Eigenvalues of A*B = {evalues4}\nEigenvalues of A * Eigenvalues of B = {evalues * evalues2}')
     
    #OUTPUT:
    '''
    Eigenvalues of A = [4. 2.]
    Eigenvalues of B = [2. 3.]
    Eigenvalues of A+B = [4. 7.]
    Eigenvalues of A + Eigenvalues of B = [6. 5.]
    Eigenvalues of A*B = [ 3.22799813 11.77200187]
    Eigenvalues of A * Eigenvalues of B = [8. 6.]
    '''
