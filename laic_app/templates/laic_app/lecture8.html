  • Augmented Matrix for a system of equations Ax = b: [A b]
  • Recall that Ax = b is always solvable as long as b is in the column space of A
  • If any combination of the rows of A gives 0, that same combination of the rows of b must give 0
  • To find a particular solution x to Ax = b, set all free variables to 0 and solve for the pivot variables
  • The complete solution to Ax = b is the x particular + all combinations of the special solutions of the nullspace
  • Take an m x n matrix A. If the rank of A = n, then there are no free variables so the nullspace only contains the 0 vector and the complete solution x = x particular if x particular exists (0 or 1 solution)
  • Take an m x n matrix A. If the rank of A = m, then there are n - r free variables and Ax = b can be solved for every b (infinite solutions)
  • Take an m x n matrix A. If the rank of A = m = n, then the matrix is invertible, so the rref form is the identity matrix and Ax = b can be solved for any b (1 solution)
  • Take an m x n matrix A. If the rank is less than m and n, then there are 0 or infinite solutions
