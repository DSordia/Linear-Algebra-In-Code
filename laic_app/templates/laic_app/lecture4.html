  • Inverse of a product: (A*B)-1 = B-1*A-1
  • Transpose of a matrix: rows become columns, columns become rows
  • Transpose of the identity matrix is the identity matrix
  • Transpose of A*A-1 is (A-1)T*AT
  • The inverse of AT is (A-1)T
  • Recall that in each step of elimination, an elementary matrix E is produced. Take a 3x3 matrix A. After elimination, we have E32*E31*E21*A = U. To get the form A = LU, we do A = E21-1 * E31-1 * E32-1 * U
  • For A = LU (assumes no row exchanges), L is lower-triangular and U is upper-triangular
  • The inverse of an elementary matrix is that same elementary matrix with the non-0 / non-diagonal value flipped
  • Easier way to generate L: For each step of elimination, place the multiplier directly into L (assuming no row exchanges)
  • For an n x n matrix A, the number of operations to complete elimination (with no row exchanges) is about (1/3)n3, and the number of operations for the column vector b is n2
  • Let P be a permutation matrix. Then P-1 = PT
  • The number of permutations of an n x n matrix is n!

    # All row permutations of a 3x3 identity matrix
     
    I = np.identity(3)
    permutations = []
    permutations.append(I)
     
    def permute(A):
        P = np.random.permutation(A)
        if len(permutations) == math.factorial(3):
            return permutations
        for permutation in permutations:
            if (P == permutation).all():
                return permute(A)
        permutations.append(P)
        return permute(A)
     
    permute(I)
     
    print('All permutations of 3x3 Identity matrix: ')
    for permutation in permutations:
        print(f'{permutation}\n')
