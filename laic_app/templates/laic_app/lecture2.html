  • Elimination: The most common method of solving systems of linear equations
  • Process for elimination: Put the coefficients of the equations in a matrix A and perform row operations (subtract a multiple of one row from another row) until you have all zeroes on the bottom-left triangle under the diagonal of the matrix
  • Perform the same row operations on the solution vector b, now solve using back-substitution - solve for z, plug this in to solve for y, then plug y and z in to solve for x
  • If you have a 0 in a pivot position (the diagonal of the matrix), switch that row with a row below it which doesn't have 0 in that position. If a row exchange can't be done, then elimination fails
  • Identity Matrix: All 0s with all 1s across the diagonal - the linear-algebra-equivalent of the number 1 (multiplying by it does not change the matrix)
  • Matrix * Column = Column, Matrix * Row = Row
  • Each step in the elimination process has an associated Elementary Matrix E, where multiplying A * E gets you the matrix with the eliminated value
  • Matrix multiplication is associative: E1 * (E2 * A) = (E1 * E2) * A
  • Permutation Matrix (P): A type of elementary matrix which permutes (flips) the rows or columns of a matrix. P can be determined by permuting the rows of the Identity matrix.
  • To flip the rows of a matrix A: P * A, to flip the columns of a matrix A: A * P
  • Matrices are not commutative by multiplication: A * B does not equal B * A

            3*x + 8*y + z - 12,

    tempA = str(tempA); tempb = str(tempb)

    A = np.array(tempA)
    b = np.array(tempb)
    x = np.linalg.solve(A, b)

    # Permuting the rows of a 2x2 matrix
     
    A = np.array([[1, 2], [3, 4]])
     
    def permute(A):
        P = np.random.permutation(A)
        if (P == A).all():
            return permute(A)
        else:
            return P
     
    print(f'A = \n{A}\nA after permuting rows = \n{permute(A)}\n')
     
    #OUTPUT:
    '''
    A =
    [[1 2]
     [3 4]]
    A after permuting rows =
    [[3 4]
     [1 2]]
    '''
     
    # Permuting the columns of a 2x2 matrix - method taken from https://stackoverflow.com/a/50381558/12733759
    A = np.array([[5, 6], [2, 3]])
     
    originalA = np.copy(A)
     
    col1 = 0; col2 = 1
    A.T[[col1, col2]] = A.T[[col2, col1]]
     
    print(f'A = \n{originalA}\nA after permuting columns = \n{A}\n')
     
    #OUTPUT:
    '''
    A =
    [[5 6]
     [2 3]]
    A after permuting columns =
    [[6 5]
     [3 2]]
    '''
