Linear Algebra In Code

Gilbert Strang's MIT 18.06 Linear Algebra Lecture Series - In Code!

This web application was built to put the concepts presented by Professor Gilbert Strang in the MIT 18.06 Spring 2005 Linear Algebra lecture series into code. This will help one see the concepts presented in another way and demonstrate the power of Python's NumPy and SymPy libraries.

Each lecture page consists of the lecture video from YouTube, the key ideas from the lecture, and code showing the concepts applied in the Python programming language.
