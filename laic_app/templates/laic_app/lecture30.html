  • Transformations: a transformation T is a function that maps an input v to an output T(v)
  • Linear transformations have the following rules:
    T(v+w) = T(v) + T(w)
    T(cv) = cT(v)
    T(cv + dw) = cT(v) + dT(w)
    T(0) = 0
  • For matrices, T(v) = Av
  • If you know T(v1), T(v2), ..., T(vn) then you know what T does for any v with basis v1, v2, ..., vn, since v = c1v1 + ... + cnvn which implies T(v) = c1T(v1) + ... + cnT(vn), where the coordinates of v are c1, ..., cn
  • Let v1, ..., vn be an input basis and w1, ..., wn be an output basis. Then the columns of A are the coefficients of a where T(v1) = a11w1 + a21w2 + ... + am1wm, T(v2) = a 12w1 + ... + am2wm
