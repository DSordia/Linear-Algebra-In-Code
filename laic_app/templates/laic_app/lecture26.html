  • Take a complex vector z, where each element of the vector is a complex number. Then zTz is does not exist - use z̅Tz for the length of the vector, where z̅ is the conjugate of z. zHz is equivalent, where H represents taking both the conjugate and transpose at the same time
  • Recall that symmetric matrices are when AT = A. For complex matrices, this is not true; AH = A is true
  • Hermitian matrices: symmetric complex matrices. Has real eigenvalues and orthogonal eigenvectors. Each eigenvector with unit length makes up a matrix Q. With real matrices we had QTQ = I, however with hermitian matrices we have QHQ = I. This Q is called a unitary matrix
  • Fourier matrix: an n x n hermitian matrix F with the first column having all 1s, and each subsequent column starting with 1 and continuing with w, w2, ..., wn-1 and then w2, w4, ... w2(n-1), ... wn-1, w2(n-1), w(n-1)2
  • wn = 1 and w = ei * 2 pi / n, which implies w is some value on the unit circle of the complex plane for any given angle 2 pi / n
  • Fast Fourier Transform: connects an n x n fourier matrix to an n / 2 x n / 2 fourier matrix; multiplies an n x n matrix in 1/2*n*log2n steps, rather than 2n steps. Used mainly in signal processing

    import numpy as np
     
    # Magnitude of a complex vector
     
    complexvect = np.matrix([[1], [1j]])
     
    print(f'Complex vector = \n{complexvect}\nMagnitude of vector = {np.dot(complexvect.getH(), complexvect)}')
     
    #OUTPUT:
    '''
    Complex vector =
    [[1.+0.j]
