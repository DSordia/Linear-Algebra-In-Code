  • Markov matrices: all values in the matrix are greater than or equal to 0, and the sum of all of the values in the columns is 1
  • For markov matrices, Î» = 1 is always an eigenvalue, and the magnitude of all other eigenvalues is less than or equal to 1
  • The steady state of a markov matrix is only the first element c1x1, where all values in the x1 eigenvector are greater than or equal to 0, since the rest approach 0
  • Take a vector v = x1q1 + x2q2 + ... + xnqn where q1, ..., qn is an orthonormal basis. Then xi = qiTv
  • Fourier series: f(x) = a0 + a1cos(x) + b1sin(x) a2cos(2x) + b2sin(2x) + ... this is an infinite-dimensional subspace of functions, where the basis is the coefficients on each a and b
  • Dot product of functions f and g: fTg = the integral of f(x)g(x)dx with bounds 0 to 2pi

    import numpy as np
     
    # Eigenvalues/eigenvectors of a markov matrix
     
    A = np.array([[.9, .2], [.1, .8]])
    evalues, evectors = np.linalg.eig(A)
     
    print(f'Markov matrix A = \n{A}\nEigenvalues of A = {evalues}\nEigenvectors of A = \n{evectors}')
