  • If A is an m x n matrix with m < n, then there are nonzero solutions to Ax = 0, because there will be at least n - m free variable(s)
  • Independence: vectors in a vector space are linearly independent if no combination gives the zero vector
  • The columns of a matrix A are linearly independent if the nullspace of A only contains the zero vector, otherwise they are dependent
  • If all n columns of a matrix are linearly independent, then the rank is n, otherwise rank < n
  • Span: vectors span a space if the space consists of all combinations of those particular vectors (i.e the columns of a matrix span the column space)
  • A basis for a vector space is a sequence of vectors all of which are linearly independent and span the space
  • For an n-dimensional space, every basis contains n vectors - n is the dimension of the space
  • For an n x n matrix, the columns of the matrix can only form a basis if the matrix is invertible
  • The number of pivot columns of a matrix = the rank of the matrix = the dimension of the column space = the number of independent vectors needed to form a basis
  • The number of free columns of a matrix = the dimension of the null space
