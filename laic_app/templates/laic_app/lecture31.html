  • Image compression: Images (and videos) are composed of individual pixels. The standard basis to account for this is that each pixel is given a color value. Compression (such as JPEG) breaks images into groups of 8x8 pixels, where the optimal basis (from the Fourier Basis or Wavelets Basis) is applied to each group, where each pixel in the group is given a coefficient. Next, any coefficients that are below a certain threshold are thrown out, which reduces each group to even less pixels
  • For videos, you can perform compression on each individual frame of the video, where frames that are nearly identical (right after one another) are thrown out
  • Wavelets Basis: a basis that is as effective as the fourier basis. To find the coefficients for a group of pixels p = p1, ..., p8 can be written as a linear combination of the wavelet vectors c1w1 + ... + c8w8 which implies p = WC which implies C = W-1p
  • To change the basis to the wavelets basis, where x is the old basis and c is the new basis, there is some matrix W such that x = Wc
  • Let T be a linear transformation. Then there is some matrix A with respect to a basis v1, ..., v8 and some matrix B of the wavelets basis w1, ..., w8 where the original basis x = c1v1 + c2v2 + ... + c8v8 which implies T(v) = c1T(v1) + c2T(v2) + ... + c8T(v8) and the columns of matrix A consist of the coefficients of the output vector T(vi)
  • Two matrices A and B produced by the same linear transformation are similar, which implies B = M-1AM where M is the change of basis vector
