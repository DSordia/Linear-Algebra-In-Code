  • Take Matrices A, B, and C, where A * B = C. Any element in the ith row and jth column of C can be found by summing up the product of multiplying each element in the ith row of A by each element in the jth column of B.
  • Rule of matrix multiplication: Take a matrix A with dimensions m x n and a matrix B with dimensions a x b. To multiply A and B, n must equal a (The number of columns of the first matrix must equal the number of rows of the second)
  • If you multiply matrix A with dimensions m x n by matrix B with dimensions n x p, the solution matrix will have dimensions m x p
  • If A * B = C, each column in C is generated by multiplying A by each column in B (The columns of C are combinations of the columns of A)
  • If A * B = C, each row in C is generated by multiplying B by each row in A (The rows of C are combinations of the rows of B)
  • If A * B = C, C is the sum of the columns of A times the rows of B.
  • Inverses of Matrices: If a matrix has an inverse, it is said to be invertible or non-singular
  • Take a square matrix A. If it has an inverse A-1, then A * A-1 = I and A-1 * A = I (Multiplying a matrix by its inverse equals the identity matrix)
  • A matrix A is singular or not invertible if: its determinant is 0, there is no matrix it can be multiplied by to generate the identity matrix, or if there exists a non-zero vector x such that A * x = 0 (any of the columns of A are a multiple of another column).
  • Gauss-Jordan method for finding the inverse: Take A * A-1 = I. Construct an augmented matrix A | I by attaching I to the right side of matrix A. Now perform elimination on this augmented matrix until A becomes I ( A | I --> I | P ) Whatever the original I became (P) is the answer A-1
