  • To find the determinant of a 3x3 matrix, break it into 6 separate matrices where each has all zeros except one value in each row, the determinant of each of these is the product of the values in each row. The answer is the sum of all these determinants (note that the determinant will be negative if it only takes one flip to return to a diagonal matrix)
  • General formula for the determinant of an n x n matrix: the sum of n! terms + or - a1ia2ja3k...anl where i, j, k, ..., l are permutations of 1, 2, 3, ..., n
  • Cofactors: pick one row or column and multiply the first element of that row or column by the determinant of the matrix formed by all of the values outside the column and row of the value you started with. Keep doing this for each element in the row or column you initially picked, rotating between adding and subtracting the product from each iteration. (add if the number of the row + number of the column is even, subtract if the number of the row + number of the column is odd) Note that if the matrix is bigger than 3x3, you will have to do cofactors within cofactors, etc..
  • Another way of describing cofactors: take a matrix A with i rows and j columns. Then the cofactor Cij is aij + or - det(n - 1 matrix excluding row i and col j) (+ if i + j is even, - if i + j is odd)
  • Cofactor formula (A is a n x n matrix, C is a cofactor of A): det A = a11C11 + a12C12+...+a1nC1n for row 1, and continue for the rest of the rows

    # LECTURE 19 - Determinant Formulas and Cofactors
     
    import numpy as np
     
    # Not too much code for this one.. cofactor expansion is done behind the scenes
     
    # Determinant of a 4x4 matrix
     
    A = np.array([[0, 0, 1, 1], [0, 1, 1, 0], [1, 1, 0, 0], [1, 0, 0, 1]])
    B = np.array([[1, 1, 0, 0], [1, 1, 1, 0], [0, 1, 1, 1], [0, 0, 1, 1]])
    print(f'A = \n{A}\nDeterminant of A = {np.linalg.det(A)}')
    print(f'B = \n{B}\nDeterminant of B = {np.linalg.det(B)}')
